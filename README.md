This is just a small program to fix a issues I had with cumulusMX not outputting pressure data to the WXNOW.txt file. As many will know the code behind that files generation is very very VERY old, and has lots of bugs
instead of trying to fix it I just made a program that will take the data from the new and reliable realtimegauges.txt file and make it into the same old format for software like APRS. Its not perfect as 24h rainfall data
is saved in the program I made. But it will save it persistently so if you restart this program it will save the data. Let me know if there are any bugs that need fixing. Ill keep this updated as I find and correct bugs.
I hope this helps atleast 1 person who has the same issue I had.
